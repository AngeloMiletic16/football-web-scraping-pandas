# Football Web Scraping with Pandas

## Project Overview

This mini project demonstrates how to collect and organize football match data using Python and pandas.
The data is obtained directly from the website **football-data.co.uk**, which provides historical match statistics in CSV format.

The goal of the project is to practice working with real-world data by reading datasets from URLs, organizing them into pandas DataFrames, and combining multiple leagues and seasons into a structured dataset.

## Technologies Used

* Python
* Pandas
* Jupyter Notebook

## Data Source

The data used in this project comes from:

https://www.football-data.co.uk/

This website provides downloadable CSV files containing match results and statistics for multiple football leagues.

## What This Project Does

* Reads football match data directly from a URL using pandas
* Loads multiple CSV files from different leagues and seasons
* Organizes the datasets into a dictionary of pandas DataFrames
* Cleans and renames selected columns
* Combines multiple datasets into one structured dataset

## Example Data

The dataset contains match information such as:

* League
* Date
* Home Team
* Away Team
* Home Goals
* Away Goals
* Match Result

## Project Structure

The main analysis and data processing are performed in the Jupyter Notebook:

`web_scraping_pandas_football.ipynb`

The notebook demonstrates how to retrieve, process, and organize football match data using pandas.

## Purpose of the Project

This project was created as a learning exercise to practice:

* working with data from web sources
* data manipulation with pandas
* organizing datasets programmatically

It is part of a broader learning path focused on data analysis and Python.
